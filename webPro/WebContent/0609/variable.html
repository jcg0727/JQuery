var : funtion-scoped(함수범위)
변수 선언과 초기화(메모리할당) 가 동시에 이루어진다
변수를 재선언해도 에러가 나지 않고 각기 다른 값을 출력한다.
유연한 변수 선언 방식으로 코드량이 많아질 경우에는 값이 변경되는 상황이 발생할 수 있다.

이것을 보완하기 위해 es6이후 추가된 변수 선언 방식이 1et과 const이다.

1et : block-socped(블럭 범위)
변수 사용전까지 초기화 되지 않는다.
변수 재할당이 가능하지만 재 선언은 불가하다.

const : block-scoped(블럭범위)
변수 사용전까지 초기화되지 않는다.
변수 재할당과 재선언이 불가하다.

let, const의 차이점음 immutable(불변)이다.
