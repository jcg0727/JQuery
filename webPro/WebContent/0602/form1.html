input type = "button" value="전송"은 정송을 할 수 없다.
전송하기 위해서 javascript를 이용해 기능을 부여해야 한다.
value속성을 반드시 기술한다.

기능을 부여하기 위해 input태그에 onclick을 설정하여 함수 이름을 호출한다.
input type = "button" value="전송" onclick="함수이름()"
