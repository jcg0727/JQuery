charAt(index) : index번째에 있는 문자를 리턴
charCodeAt(index) : index번째에 있는 문자의 unicode를 리턴 A -> 65 a -> 97 0 -> 48

indexOf(value, index) : value값의 위치를 리턴, index는 검색 시작위치
lastIndexOf(value, index)

trim() : 앞뒤의 공백을 제거
text입력 후 script로 보내기 -value

start ~ end 사이의 문자열을 잘라내기, end는 포함되지 않음
substring(start, end)
slice(start, end)
substr(start, length)

split('구분자','갯수') : 구분자를 기준으로 문자열을 분리하여 배열로 리턴

replace(old, new) : old문자를 new문자로 바꾼다
textarea에 입력 - value를 이용하여 스크립트로 보내기 - \r\n이 포함되어 있다
div에 출력 - innerHTML을 이용하여 출력 - \r\n을 <br>태그로 바꾸어서 출력

div에 입력되어 있는 문자를 스크립트로 가져가기 - innerHTML - <br>이 포함
textarea에 출력 - value - <br>태그를 \r\n으로 변경
