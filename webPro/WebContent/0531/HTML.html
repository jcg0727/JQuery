각 줄의 끝에서 줄을 바꾸려면 <br>태그를 이용 
  p태그는 줄이 끝나면 자동으로 줄이 바뀌며 줄 사이에 여백이 생긴다

  머릿글 헤더를 나타내는 태그 h1 ~ h6
  페이지 이동 태그 : <a href = "">	
  a태그의 href 속성 : 이동할 페이지의 주소나 이름

  이미지를 나타내는 태그 :<img src = "" alt = "사진">
  이미지를 나타내기 위한 속성(src)과 이미지를 대체할 속성(alt)이 있다

  html의 태그에 스타일을 부여하기 위해 css가 필요
  내부 css - html문서안에<style></style>
  외부 css - 별도의 css 문서를 작성해서 html 문서 내부에 파일로 삽입
  <link rel = "stylesheet" href="./css/outStyle.css">

  내부 css와 외부css에 같은 요소가 중복되어 있을 때 내부 css가 우선으로 실행
